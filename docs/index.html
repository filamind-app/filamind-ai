Filamind AI Synology Package Source

Private, local AI chat for Synology NAS. This page is a third-party DSM Package Source — point Package Center at it for one-click updates.

🚀 One-click updates via DSM Package Center

  1. Open DSM → Package Center → Settings → Package Sources → Add
  2. Name: Filamind AI
  3. Location: https://filamind-app.github.io/filamind-ai/
  4. Click OK. DSM verifies the source.
  5. Switch to the Community tab — Filamind AI appears with the current version and an Install button.

After every release on GitHub, DSM detects the new version and offers an Update button.

📥 Manual install (alternative)

Pick the SPK matching your device from the Releases page and install via Package Center → Manual Install.

🖥️ Supported devices

  • DVA 3221 (NVIDIA GTX 1650 4GB, CUDA 10.1) — GPU-accelerated llama.cpp b1620, supports Llama 2 / Mistral v0.2 / Phi-2 / CodeLlama / AceGPT (Arabic)
  • DS1821+ (AMD Ryzen V1500B, AVX2 CPU) — Latest llama.cpp, supports Llama 3.x / Qwen 2.5 / Gemma 2/3 / Phi-3/4 / Aya Expanse / Mixtral / 70B-class models
