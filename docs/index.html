Filamind AI Synology NAS

Private, local AI chat for Synology NAS — runs LLMs on your own hardware.

📥 Recommended: manual install

Download the SPK matching your device from the Releases page and install via DSM → Package Center → Manual Install. This is the reliable path today.

  • DVA 3221 (GTX 1650, CUDA): Filamind-X.Y.Z-dva3221.spk
  • DS1821+ (CPU): Filamind-X.Y.Z-ds1821-plus.spk — ⚠️ inference not yet functional (build-stub binary)

🔄 One-click updates via Package Center experimental — not working yet

Honest status: a DSM Package Source must answer the POST request Package Center sends with a filtered package list. GitHub Pages is static and returns HTTP 405 to POST, so adding https://filamind-app.github.io/filamind-ai/ as a source does not work — DSM will reject it with "Invalid location".

Making it work needs a small dynamic backend (e.g. a Cloudflare Worker or any server-side script) that reads DSM's POST params and returns packages.json filtered by device. That backend is planned but not yet deployed. Until then, use manual install above and watch the Releases page (or the in-app Settings → About → Check for updates button, which works today).

🖥️ Supported devices

  • DVA 3221 (NVIDIA GTX 1650 4GB, CUDA 10.1) — GPU-accelerated llama.cpp b1620, supports Llama 2 / Mistral v0.2 / Phi-2 / CodeLlama / AceGPT (Arabic)
  • DS1821+ (AMD Ryzen V1500B, AVX2 CPU) — Latest llama.cpp, supports Llama 3.x / Qwen 2.5 / Gemma 2/3 / Phi-3/4 / Aya Expanse / Mixtral / 70B-class models
