# Filamind AI · Package Source

This page **is** a third-party Package Source for Synology DSM.

## One-click updates via DSM Package Center

1. Open **DSM → Package Center → Settings → Package Sources → Add**.
2. Name: `Filamind AI` · Location: `https://filamind-app.github.io/filamind-ai/`
3. Click **OK**. DSM verifies the source and adds it.
4. Switch to the **Community** tab in Package Center — Filamind AI now appears with the current version.
5. After every release on GitHub, DSM detects the new version and offers an **Update** button.

## Manual install

Direct SPK downloads per device live on the [Releases](https://github.com/filamind-app/filamind-ai/releases) page.
